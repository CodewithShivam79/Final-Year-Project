# Online Book Store PHP

Simple PHP-based online bookstore project.

## Project structure

- `obsms/` - frontend pages, assets, CSS, JS, fonts, vendor libraries
- `obsms3/` - admin area and alternate frontend files
- `.gitignore` - ignored files for Git

## Installation

1. Copy the repository to your local web server folder.
2. Use XAMPP / WAMP / LAMP / MAMP.
3. Open the project in your browser via `http://localhost/...`.

## Notes

- Keep any sensitive files like `.env` out of Git.
